# Chrome Web Store Listing Draft

## Extension Name

Task Assistant

## Short Description (132 chars max)

Detect email deadlines fast, prioritize urgent tasks, and add reminders to Google Calendar in one click.

## Detailed Description

Task Assistant helps you quickly find what is due and what to do next.

### What it does

- Extracts deadline-related tasks from recent Gmail emails.
- Groups deadlines into Overdue, Today, Tomorrow, and This Week.
- Highlights risk levels (Overdue, Due <24h, Due <3d, Upcoming).
- Lets you open source emails directly in Gmail.
- Adds reminders to Google Calendar in one click.
- Provides optional concise AI summaries using your Gemini API key.

### Why users like it

- Deadline-first view for faster decision making.
- Minimal workflow: Open in Gmail, Add to Calendar, Done.
- Focus mode: Show only high-risk tasks.

### Permissions explanation

- `gmail.readonly`: used to read emails and detect deadlines.
- `calendar.events`: used to create reminders when the user requests it.
- `identity`: used for secure Google OAuth login.
- `storage`: used to save local extension settings and task state.
